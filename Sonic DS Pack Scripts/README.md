# Sonic DS Music Pack Scripts

These are the scripts for my Sonic DS Music Pack. Because the mod on its own contains so many music files which could bloat out the repository and cause the download speeds to slow to a crawl when cloning it, I've only included the scripts here.

If you want the mod itself, you can find it on Gamebanana here: https://gamebanana.com/mods/442450

You can expect these scripts to be either spun off into their own mod or integrated into Even More Themes.
